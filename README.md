# Sales-Dashboard

## Project Overview
This Power Bi dashboard analyzes sales performance and through interactive visuals.

## Objectives
- Analyze overall sales performance
- Compare sales across regions
- Track key KPIs such as:
  - Total Sales
  - Average Order Value
  - Number of Orders

## Dataset
- <a href="https://services.odata.org/Northwind/Northwind.svc/">Dataset</a>

## KPIs
- Compare sales and orders 
- Which month got highest sales
- Which region is contributing maximun to sales
- Highest selling category
- Top 5 companies contributing to sales
- Trend analysis over time

## Tools & Technologies Used
- Power BI Desktop
- Power Query
- DAX in Power BI
